欢迎使用 Agnes AI

首次使用需要先配置 API Key。完成下面三步后，回到软件全局设置里粘贴并保存。
1

注册 Agnes AI

访问 Agnes AI 官网，按页面提示完成注册。注册完成后登录账号。

打开 agnes-ai.com
2

获取 API Key

进入 API 平台页面，创建并复制你的 API Key。请不要把 Key 发给别人。

前往 API 平台
3

填入全局设置

回到软件，点击下方按钮打开全局设置，将 API Key 粘贴到输入框并保存。

打开全局设置填写 API Key
配置完成后再次启动软件就不会再弹出这个窗口。视频生成和提示词扩写共用同一个 API Key。
